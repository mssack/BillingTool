###RC = Release Candidate
+  Die Release Kandidaten können verwendet werden, Sie sind aber mit Vorsicht zu genießen. 
+  Die meisten RC's werden sollten Sie nicht verwendet werden wieder gelöscht werden.

####Inhalt
+ Die aktuellen Executeables
+ Die Datenbankdefinitionen.
+ Die enumerationen die für das Entwickeln von Software nützlich sein könnten
+ Batch scripts die für das Testen und Verstehen herangezogen werden können.
